# Руководство по Деплою: Bloomberg Terminal Elite

Чтобы твои друзья могли пользоваться приложением, а данные обновлялись автоматически в облаке, лучше всего использовать **Streamlit Community Cloud**. Это бесплатно и надежно.

## Шаг 1: Подготовка файлов
Убедись, что в папке проекта есть следующие файлы:
1.  `app.py` (главный файл)
2.  Все модули (`market_data.py`, `news_feed.py`, `cot_dashboard.py` и т.д.)
3.  `requirements.txt` (я его уже создал, он содержит список библиотек)

## Шаг 2: Создание репозитория на GitHub
1.  Создай новый репозиторий на [GitHub](https://github.com/new).
2.  Загрузи все файлы проекта в этот репозиторий.
    - Можно через Git: `git init`, `git add .`, `git commit`, `git push`.
    - Или просто перетащив файлы через веб-интерфейс GitHub.

## Шаг 3: Деплой на Streamlit Cloud
1.  Зайди на [share.streamlit.io](https://share.streamlit.io).
2.  Нажми **"New app"**.
3.  Выбери свой репозиторий, ветку (`main` или `master`) и главный файл (`app.py`).
4.  Нажми **"Deploy!"**.

## Авто-обновление данных
- Приложение работает "на лету". Каждый раз, когда кто-то заходит на сайт, оно подтягивает свежие данные через API (yfinance, RSS).
- Я настроил **кэширование (TTL)**:
    - Котировки обновляются каждую минуту.
    - Новости и макро-данные — каждые 5-10 минут.
- Тебе не нужно ничего перезапускать, всё будет "живым" 24/7.

## Кастомный домен
В настройках Streamlit Cloud ты можешь изменить URL на что-то вроде `my-bloomberg-terminal.streamlit.app`, чтобы было удобнее делиться ссылкой.
